一.File and Code Templates
（1）Java Class 注释模板配置
1）沿路径File->Settings->Editor->File and Code Templates->Includes->File Header，打开文件模板配置面板：
2）配置注释：
/**
* Title: ${NAME}
* Description:
*
* @author Fang Hongyu
* @date ${DATE} ${TIME} 首次创建
* @date ${DATE} ${TIME} 最后修改
* @version 1.0
* @copyright 中科软科技股份有限公司
*/
3）左键单击OK按钮保存。
一.Life Templates
（1） Java Class 注释模板配置
1）沿路径File->Settings->Editor->Life Templates，打开实时模板（Live Templates）配置窗口：
2）左键单击红框中的按钮弹出菜单：
3）选择Template Group选项，弹出对话框：
4）在红框中输入模板组名称：JavaAnnotation，左键单击OK按钮添加模板组JavaAnnotation完毕：
5）选定JavaAnnotation模板组，左键单击红框中的按钮弹出菜单：
6）选择Live Template选项，弹出面板：
7）左侧红框中输入触发自动注释的前缀：aclass，右侧红框中输入注释描述：java class annotation，下侧红框中填充注释模板内容：
/**
 * Title:$className$
 * Description:
 * 
 * @author Fang Hongyu
 * @date $date$ $time$  首次创建
 * @date $date$ $time$  最后修改
 * @version 1.0
 * @copyright 中科软科技股份有限公司
 */
8）左键单击Define按钮弹出菜单：
9）选择Java选项，左键单击Edit variables按钮弹出对话框：
10）配置红框中的表达式（Exception）后左键单击OK按钮配置变量date、time完毕，最后左键单击红框中的OK按钮完成Java Class 注释模板配置。
